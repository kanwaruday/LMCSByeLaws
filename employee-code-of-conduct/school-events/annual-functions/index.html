Annual Functions

This office order provides comprehensive guidelines for organising the Annual Function at La Montessori Schools, covering preparation, timelines, themes, job roles, and event-day setup.

📋

A. Preparation & Committees

Performance quotas, class preparation guidelines, and organising committee roles
📅

B. Timelines

Theme selection by March, dresses by 60 days prior, complete countdown checklist
🎨

C. Themes, Songs & Dress

Theme approval process, performance selection criteria, event flow structure, costume guidelines
🔄

D. Changes for Preparation

Daily zero period from 30 working days prior, classroom setup, compering team formation
👔

E. Job Roles

Principal, IT Teacher, Class Teachers, Activity Teacher, PTI, Clerk cum PRO, English TGT/PGT responsibilities
🏫

F. Annual Function Setup

Duty roster guidelines for entry, student areas, green house travel, transport, food, and corridor discipline
