B. Timelines

Key Deadlines

  • Theme Selection: Before last week of March (New Year Preparation week)
  • Performance & Song Selection: 31st March each year
  • Annual Function Preparation: April month & 30 working days before the Annual Function
  • Event Flow: 60 working days before Annual Function
  • Dress Finalisation: 60 working days before Annual Function
