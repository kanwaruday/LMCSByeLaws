Events SOP

Subject: Standard Operating Procedure and Timelines for Conduct of School Events

To ensure uniformity, timely execution, and financial transparency during all events organised under the La Montessori Chain of Schools, the following SOP and timelines must be strictly adhered to by all schools.

A. Event Planning & Approval Timelines

Stage Action / Activity Responsibility Timeline (Before Event)
1 Event Proposal and SOP Format filled (Event Name, Date, Target Students, Mode, Guests, etc.) Event In-charge 2 Weeks Prior
2 Approval of Event Plan by Principal and email to System Coordinator Principal / System Coordinator Within 2 days of SOP submission
3 Mock Drill 1 (Qualifying Round) — check flow, resources, and student readiness Event In-charge / Teachers 7 Days Prior
4 Improvement Round 1 based on feedback Event In-charge 6 Days Prior
5 Mock Drill 2 (Final Rehearsal) — check overall readiness Event In-charge / Admin Manager 3 Days Prior
6 Improvement Round 2 (Final Corrections) Concerned Teachers / Support Staff 2 Days Prior
7 Final Event Day All Concerned As Scheduled
8 Post-Event Report & Profit–Loss Sheet Submission Event In-charge / Admin Manager Within 2 days after event

B. Event Execution Checkpoints

  1. The Event SOP Template (as per "Event Organising Template V1") must be filled completely and submitted for approval before any preparations begin.
  2. Mock Drills are mandatory. Photographs/videos of each rehearsal should be maintained as documentation.
  3. The Admin Manager shall review all logistics, seating, electrical and safety arrangements at least two days prior to the final event.
  4. Principal's final clearance must be obtained before the event goes live.

C. Financial Collection & Accountability

  1. Collection from Students: Class Teachers collect the event amount. Collection sheet must include student names, class strength, and total amount collected.
  2. Deposit to Clerks: Class Teacher hands over total collected amount to the school clerk against a signed receipt. Clerk updates the centralised collection sheet shared by the Admin Manager.
  3. Consolidation: Admin Manager consolidates all school-wise collection data and finalises budget allocation for each school.
  4. Expense Planning: All event-related purchases must be within approved budget limits. Bills and invoices must be maintained for audit and reimbursement.
  5. Post-Event Reconciliation: Profit & Loss Statement prepared by Admin Manager and submitted to Admin (Operations) and Accounts Head within two working days. Any balance amount deposited back to the main account immediately.

D. Documentation to be Maintained

  1. Filled Event SOP Template (V-1)
  2. Mock Drill and Final Event Attendance & Feedback Sheets
  3. Budget Approval and Expense Vouchers
  4. Profit & Loss Statement
  5. Photographs/Videos of Event
Non-compliance with the timelines or SOP procedure may lead to administrative remarks during performance review or financial audit.
