Letter, Parcel, Application Receiving Process

Letter, Parcel & Application Receiving Process

In lieu of recent confusions with the documentation of applications submitted by Staff and Students at the School, the following protocol is to be followed:

Hard Copy Application Process

  1. Applications must be accepted only if they carry the signature of the applicant along with the date.
  2. All applications must be entered in the receival register present at each School.
  3. The Principal's signature must be taken on the receival register.
  4. The application must be signed by the Principal along with remarks and further action.
  5. The application must then be taken by the Fee Clerk for filing in the appropriate folder.
Note: Failing to follow the above protocol will result in the application not being accepted.
